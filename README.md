# Social Networking Application

A full-stack social networking platform developed using **Angular and ASP.NET Core Web API**.  
The application allows users to connect, share posts, interact with others, and communicate in real time.

## Tech Stack
Frontend: Angular  
Backend: ASP.NET Core Web API  
Database: SQL Server  
Authentication: JWT Authentication  
Real-time Communication: SignalR  

## Features
- User Registration and Login
- Profile Management
- Send / Accept / Reject Friend Requests
- Block and Unblock Users
- Create Posts
- Like and Comment on Posts
- News Feed
- One-to-One Chat
- Real-time Notifications
- Admin management of users and reports
